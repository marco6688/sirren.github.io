# 命名空间 using 的使用

```c++
#include <iostream>

using std::cin;     //当我们使用cin时，从命名空间std中获取

int main(){

    int i;
    cin >> i;       //正确，通过using声明
    cunt << i;      //错误，没有声明，必须使用完整名称
    std::cunt << i; //正确，从std中使用cout
    return 0;

}

```

- 每个名字都需要独立的using声明
- 头文件中不应包含using声明

# 标准库类型 string

标准库类型string是可变长点string的字符序列，使用string类型必须好汉string头文件

```c++

#include <string>
using std::string;

```

## 定义和初始化string

- 使用等号初始化一个变量是*拷贝初始化*
- 不使用等号则是*直接初始化*

```c++

string s1;              // 默认初始化，s1是一个空串
string s2 = s1;         // 拷贝初始化，s2是s1的副本
string s3 = "hahaa";    // 拷贝初始化，s3内容是字面值“hahaa”
string s4(10,c);        // 直接初始化，s4的内容是10个“c“ 


```

# 标准库类型 vector

标准库*vextor*表示类的集合，其中所有对象的类型都相同，集合中每个对象都有一个与之对应的索引，索引用于访问对象

## vector的使用

```c++

#include <vector>
using std::vector

```

C++ 既有*类模板*也有函数模版，其中vector是一个类模板

## 初始化与赋值

|代码|介绍|
|---|---|
|vector<T> v1|v1是一个空的vector，潜在元素是T类型|
|vector<T> v2(v1)|v2包含有v1的所有元素的副本|
|vector<T> v2=v1|等同于v2(v1)|
|vector<T> v3(n,val)|包含n个重复元素，值是val|
|vector<T> v4(n)|重复了n个初始化对象|
|vector<T> v5{n1,n2,n3...}|包含了n个初始化对象，每个元素都有相应的初始值|
|vector<T>  v6 = {n1,n2,n3...}|等同于v5{n1,n2,n3...}|

|++iter|指向iter上一个元素|

|iter1!=iter2|判断两个迭代器是否指向同一个元素|

|iter+=n|迭代器的加法复合赋值语句|

# 数组

数组与vector相同的是数组也是存放同类型对象的容器，这些对象没有名字，需要通过其所在位置访问。不同的是数组的大小确定不变，不能随意向数组中添加元素。

## 定义及初始化

数组是一种复杂数据类型，声明方式a[d],其中a是数组名称，d是维度。维度是说明数组中元素个数，由于数组大小不变所以维度是常量表达式。

```c++

unsigend cnt =42;			//不是常量表达式
constexpr unsigend sz =42;	//是常量表达式
int arr[10];					//含有10个整数的数组
int *parr[sz];				//含有sz个整型指针的数组
string bad[cnt];				//错误，cnt不是常量
string strs[get_size()];	//取决于get_size是不是常量

```

### 显式初始化

```c++

int a1[3]={1,2,3};	//初始化为维度3的数组
int a1[]={1,2,3};	//维度3的数组
int a1[5]={1,2,3};	//等价于a1[]={1,2,3，0，0};
int a1[2]={1,2,3};	//错误，初始值过多

```

- 默认情况下数组不允许拷贝

### 指针和数组

- 使用数组时，编译器一般会把他转换成指针
- 使用数组名称的时候，伴一起自动把数组转换成另一个指向数组首元素的指针

```c++

int ia[]={1,2,3,4,5,6,7,8};
int *p1 = &ia[0];					//赋值数组第一个指针
int *p2 = ia;							//同p1
auto p3(ia);							//同p1
auto p3(&ia[0]);						//同p1
p3=42;									//错误，指针类型不能赋整数值

```

### 指针也是一个迭代器

- 首元素可以通过数组名字和数组中的首地址得到
- 尾元素可以通过取一个不存在的下标来获取
- c++11引入了begin和end函数。可以获得首尾元素指针

# 多维数组

C++语言中没有多维数组，多维数组是指数组的数组

## 多维数组的初始化

```c++

//初始化一个多维数组
int a[3][3] = { {1,2,3},{4,5,6},{7,8,9} };
//同上
int b[3][3] = {1,2,3,4,5,6,7,8,9};
//初始化一个数组，每组第一个赋值，其他默认初始化
int c[3][3] = { {1},{2},{3} };
//初始化一个数组，第一组有值，其他默认初始化
int d[3][3] = {1,2,3};

```
